1. 제품 개요

  • 고정밀 ECG 센서로 동물의 심전도 측정
  • 반려동물, 실험동물 등 다양한 동물에 적용 가능
  • 블루투스 5.0 무선 통신으로 실시간 데이터 전송
  • 비침습적 측정으로 동물에 스트레스 최소화
  • C Type 충전식 배터리
  • 전문 분석 소프트웨어 포함

2. 구성품

제품 상자를 열어 아래 구성품이 모두 들어있는지 확인해주세요.

  • VetWiz 하드웨어 본품 (1개)
  • 90cm 스냅 전극 케이블 (3개)
  • 2225H 3M 패치 전극 (30개)

3. 측정 준비

측정 전 아래 위치에 맞춰 전극을 부착합니다.

  1. SIG: 좌측 뒷쪽 발바닥
  2. GND: 좌측 앞쪽 발바닥
  3. REF: 우측 앞쪽 발바닥
  4. 각 전극을 발바닥과 밀착되도록 부착합니다.

4. 기본 기능 (Basic Features)

  • 생체 신호, 심박, 호흡수 실시간 모니터링
  • 동물 별 프로필 관리 (등록 / 수정 / 삭제)
  • 생체 신호, 심박, 호흡 측정 및 측정 데이터 확인
  • 프로필, 측정 데이터 검색 기능
  • 이상 심박 알림, 소리 및 화면 등 앱 관련 설정

각 기능의 자세한 사용 방법은 어플 사용법을 참고하시기 바랍니다.

5. 기기 충전

  1. USB 충전 케이블의 타입-C 커넥터를 기기의 충전 포트에 연결합니다.
  2. 충전 시 LED 표시등이 흰색으로 표시됩니다.
  3. LED 표시등이 꺼지면 완충 상태입니다.

6. 문제 해결

증상 원인 해결 방법
기기가 켜지지 않음 배터리 방전 타입-C 케이블로 충전하세요
블루투스 연결 실패 범위 초과 또는 간섭 기기 근처로 이동하고 다시 시도하세요
ECG 신호 없음 전극 접촉 불량 전극을 재부착하거나 젤을 추가하세요
데이터 저장 오류 저장 공간 부족 불필요한 파일을 삭제하거나 외부 저장소를 사용하세요
