VetWiz-ECG 사용설명서

동물 심전도 측정 및 분석 솔루션

Ver 1.0.0 (2025.12)
제품 개요 구성품 측정 준비 (셋업) 전극 부착 방법 애플리케이션 사용법 결과 분석 방법 퀵 가이드 (Quick Guide)

1. 제품 개요

  • 고정밀 ECG 센서로 동물의 심전도 측정
  • 반려동물, 실험동물 등 다양한 동물에 적용 가능
  • 블루투스 5.0 무선 통신으로 실시간 데이터 전송
  • 비침습적 측정으로 동물에 스트레스 최소화
  • USB 충전식 배터리
  • 전문 분석 소프트웨어 포함

2. 구성품

제품 상자를 열어 아래 구성품이 모두 들어있는지 확인해주세요.

  • Vetwiz-ECG 메인 유닛 (1개)
  • 일회용 표면 전극 (10세트)
  • 재사용 전극 클립 (3개)
  • USB 충전 케이블 (1개)
  • 사용설명서 및 퀵 가이드
  • 전용 분석 소프트웨어 설치 CD 또는 다운로드 링크
  • 동물용 고정 벨트 (S, M, L 사이즈)

3. 측정 준비 (셋업)

3-1. 기기 충전

  1. USB 충전 케이블의 타입-C 커넥터를 기기의 충전 포트에 연결합니다.
  2. 충전 시 LED 표시등이 빨간색으로 표시됩니다.
  3. 약 2시간 후 LED가 녹색으로 변하면 충전이 완료됩니다.
  4. 완전히 충전되면 8시간 연속 사용 가능합니다.

3-2. 소프트웨어 설치

  1. 제공된 설치 CD를 사용하거나 NEUROWIZTEK 공식 웹사이트에서 최신 버전을 다운로드합니다.
  2. Windows 또는 macOS에 애플리케이션을 설치합니다.
  3. 초기 설정 마법사를 따라 기기와 소프트웨어를 연결합니다.

3-3. 동물 준비

  1. 측정 30분 전부터 동물을 편안한 상태로 유지합니다.
  2. 측정 부위의 털을 약 2cm 정도 다듬거나 제거합니다.
  3. 측정 부위를 깨끗한 천으로 가볍게 닦아 청소합니다.
  4. 피부가 건조한 상태에서 측정을 진행합니다.

4. 전극 부착 방법

4-1. 표준 3전극 배치 (개, 고양이)

  1. 우측 전극 (빨강): 우측 앞다리 관절 위쪽
  2. 좌측 전극 (노랑): 좌측 앞다리 관절 위쪽
    3. img src="images/vetwiz/electrode_placement_other.jpg" class="full-width-img" style="max-width: 350px; display: block; margin: 20px auto;" alt="기타 동물 전극 부착 위치" > <
  3. 접지 전극 (검정): 우측 뒷다리 관절 위쪽
  4. 각 전극을 피부와 밀착되도록 부착합니다.
  5. 전극이 움직이지 않도록 고정 벨트를 사용합니다.

4-2. 기타 동물의 전극 배치

  • 토끼: 우측/좌측 귀 뒤쪽 부위
  • 쥐/기니피그: 앞다리와 뒷다리 내측
  • 새: 가슴 양쪽 부위

4-3. 전극 접착제 바르기

  1. 일회용 전극의 뒷면에 콘덕팅 젤을 얇게 펴 바릅니다.
  2. 전극을 측정 부위에 수직으로 눌러 부착합니다.
  3. 5초 이상 누르고 있어 충분히 접착시킵니다.
  4. 부착 후 전극이 피부와 완전히 접촉하는지 확인합니다.

5. 애플리케이션 사용법

5-1. 기기 연결

  1. 애플리케이션을 실행합니다.
  2. '기기 연결' 메뉴에서 'Vetwiz-ECG' 스캔을 선택합니다.
  3. 사용 가능한 기기 목록에서 본인의 기기를 선택합니다.
  4. 연결 성공 메시지가 표시되면 준비 완료입니다.

5-2. 측정 시작

  1. '새 측정' 버튼을 클릭합니다.
  2. 동물 정보를 입력합니다: 이름, 종류, 나이, 몸무게 등
  3. 측정 목적을 선택합니다: 정기 검진, 진단, 모니터링 등
  4. '측정 시작' 버튼을 눌러 ECG 신호 수집을 시작합니다.
  5. 화면에 실시간 ECG 파형이 표시됩니다.

5-3. 측정 진행

  • 기본 측정: 동물이 안정적인 상태에서 5분 이상 측정합니다.
  • 스트레스 측정: 약한 자극 후 반응을 측정합니다.
  • 운동 측정: 가볍운 활동 후 심박 변화를 측정합니다.
  • 측정 중 신호 강도 표시기를 확인하여 신호 품질을 모니터링합니다.
  • 신호가 약하면 전극을 다시 확인하고 필요시 재부착합니다.

5-4. 측정 저장

  1. 원하는 측정 시간에 도달하면 '측정 정지' 버튼을 클릭합니다.
  2. 측정 요약 정보를 검토합니다.
  3. '저장' 버튼을 눌러 데이터를 로컬 또는 클라우드에 저장합니다.
  4. 환자 파일에 자동으로 추가됩니다.

6. 기본 기능 (Basic Features)

6-1. 블루투스 연결

  • 기기의 전원을 켜면 자동으로 블루투스 모드가 활성화됩니다.
  • 애플리케이션의 '기기 연결' 메뉴에서 사용 가능한 Vetwiz-ECG를 검색합니다.
  • 기기를 선택하면 핸드셰이크 과정을 거쳐 연결이 완료됩니다.
  • 연결 거리는 최대 10미터 이내 (장애물 없을 시)입니다.
  • 연결 해제 시 기기의 전원이 자동으로 꺼집니다 (절전 모드).

6-2. 프로필

  • 동물별로 개별 프로필을 생성하여 측정 기록을 관리합니다.
  • 프로필에는 동물 정보 (이름, 종류, 나이, 몸무게, 의료 기록 등)가 저장됩니다.
  • '프로필 추가' 버튼으로 새로운 동물 프로필을 생성합니다.
  • 프로필 정보는 언제든지 수정하거나 업데이트할 수 있습니다.
  • 프로필 사진을 업로드하여 시각적으로 구분할 수 있습니다.

6-3. 실시간 모니터링

  • 측정 중에 실시간으로 ECG 파형을 화면에 표시합니다.
  • 신호 강도 표시기를 통해 신호 품질을 모니터링합니다.
  • 심박수 (HR)가 실시간으로 업데이트되어 표시됩니다.
  • 비정상 심박이 감지되면 경고 알림이 표시됩니다.
  • 측정 시간, 신호 상태 등이 대시보드에 표시됩니다.

6-4. 데이터 저장

  • 측정 완료 후 '저장' 버튼으로 데이터를 저장합니다.
  • 데이터는 로컬 저장소 또는 클라우드 서버에 저장할 수 있습니다.
  • 자동 저장 기능을 활성화하면 측정 종료 시 자동으로 저장됩니다.
  • 저장된 데이터에는 타임스탬프, 동물 정보, 측정 파라미터가 포함됩니다.
  • 데이터 포맷: 고유한 ECG 데이터 형식 및 표준 포맷 (CSV, PDF) 지원

6-5. 데이터 열람

  • '측정 기록' 메뉴에서 저장된 모든 측정 데이터를 확인합니다.
  • 각 측정 기록을 클릭하면 상세 정보와 ECG 파형이 표시됩니다.
  • 측정 날짜, 시간, 심박수, 분석 결과 등을 열람할 수 있습니다.
  • 이전 측정 결과와의 비교 분석이 가능합니다.
  • PDF 또는 이메일로 기록을 내보낼 수 있습니다.

6-6. 검색, 필터링, 정렬

  • 검색: 동물 이름, 측정 날짜 등으로 기록을 검색합니다.
  • 필터링: 종류별, 나이별, 측정 결과별로 기록을 필터링합니다.
  • 정렬: 최신순, 오래된순, 심박수순 등으로 정렬 가능합니다.
  • 고급 검색으로 날짜 범위, 심박수 범위, 진단 결과 등으로 세밀한 검색이 가능합니다.
  • 즐겨찾기 기능으로 자주 보는 기록을 빠르게 접근할 수 있습니다.

6-7. 설정

  • 언어: 한국어, 영어, 중국어 등 다양한 언어 선택 가능
  • 단위: 심박수 표시 단위 (BPM) 변경 가능
  • 통지: 알림 및 경고 설정 커스터마이징
  • 클라우드 동기화: 자동 백업 및 다중 기기 동기화 설정
  • 프라이버시: 데이터 암호화, 접근 권한 설정
  • 디스플레이: 화면 밝기, 테마 (다크모드 등) 설정
  • 기기 정보: 펌웨어 버전, 배터리 상태, 저장 공간 확인

7. 결과 분석 방법

6-1. 자동 분석 결과

측정이 완료되면 애플리케이션이 다음 항목을 자동으로 분석합니다:

  • 심박수 (HR): 분당 심박 수 (beats per minute)
  • P-R 간격: 심방 및 심실 전도 시간
  • QRS 간격: 심실 탈분극 지속 시간
    • img src="images/vetwiz/ecg_waveform.jpg" class="full-width-img" style="max-width: 400px; display: block; margin: 20px auto;" alt="ECG 파형 예시 및 설명" > <
  • Q-T 간격: 심실 재분극 시간
  • 심리 변이도 (HRV): 심박수 변이 지표
  • 정상성 평가: 정상/비정상 판정

6-2. 그래프 해석

  • X축: 시간 (초)
  • Y축: 전기신호 (mV)
  • P파: 심방 활성화
  • QRS군: 심실 활성화
  • T파: 심실 복원

6-3. 보고서 생성

  1. 분석 결과 화면에서 '보고서 생성' 버튼을 클릭합니다.
  2. 보고서 포맷을 선택합니다 (PDF, Excel 등).
  3. '다운로드' 또는 '이메일 발송'을 선택합니다.
  4. 생성된 보고서를 환자 기록에 첨부합니다.

6-4. 비정상 신호 대처

  • 신호 잡음이 많음: 전극을 재확인하고 동물의 움직임을 최소화합니다.
  • 신호가 너무 약함: 전극을 다시 부착하거나 젤을 추가합니다.
  • 비정상 파형: 측정을 다시 진행하거나 수의사와 상담합니다.

8. 문제 해결

증상 원인 해결 방법
기기가 켜지지 않음 배터리 방전 USB 케이블로 충전하세요
블루투스 연결 실패 범위 초과 또는 간섭 기기 근처로 이동하고 다시 시도하세요
ECG 신호 없음 전극 접촉 불량 전극을 재부착하거나 젤을 추가하세요
데이터 저장 오류 저장 공간 부족 불필요한 파일을 삭제하거나 외부 저장소를 사용하세요

9. 유지 및 보관

8-1. 일상 관리

  • 매 사용 후 기기를 부드러운 천으로 닦아서 정리합니다.
  • 물에 직접 담그지 마세요.
  • 전극 부착 부위의 남은 젤을 깨끗이 제거합니다.
  • 일회용 전극은 사용 후 버립니다.

8-2. 보관

  • 실온, 건조한 장소에 보관합니다.
  • 온도 범위: 0°C ~ 40°C
  • 습도 범위: 20% ~ 80% (비응축)
  • 장기 보관 시 월 1회 충전을 권장합니다.

8-3. 정기 점검

  • 1년마다 기술 지원팀에 검사를 의뢰하세요.
  • 재사용 전극은 6개월마다 교체하세요.
  • 측정 불일치가 발생하면 즉시 점검을 받으세요.
