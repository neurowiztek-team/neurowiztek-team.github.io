Signalwiz Mobile

모바일 기반 생체신호 모니터링 및 관리 솔루션

Ver 1.0.0
제품 개요 주요 기능 앱 설치 및 연결 사용 방법 지원 및 문의

1. 제품 개요

Signalwiz Mobile은 스마트폰으로 생체신호를 실시간 모니터링하고 관리하는 모바일 솔루션입니다.

  • 간편한 모바일 연결
  • 실시간 생체신호 시각화
  • 측정 기록 저장 및 열람
  • 휴대성과 사용 편의성 강화

2. 주요 기능

  • 블루투스 기기 연결
  • 실시간 파형 모니터링
  • 측정 데이터 자동 저장
  • 검색 및 필터링 기능
  • 알림 및 경고 설정

3. 앱 설치 및 연결

  1. 앱스토어에서 Signalwiz Mobile을 다운로드합니다.
  2. 앱을 실행하고 [블루투스] 칸에서 '기기 검색 시작'을 선택합니다.
  3. 제품 기기가 목록에 나타나면 해당 버튼을 탭하여 연결합니다.
  4. 장비를 연결한 뒤 [임피던스] 탭으로 가서 접촉상태를 확인하시면 됩니다.

초기 설정에는 초기 사용 시 등록 버튼을 눌러 등록하는 과정을 포함됩니다.

4. 사용 방법

앱 내 [모니터] 화면에서 현재 측정 중인 데이터를 실시간으로 확인할 수 있습니다.

  1. 화면 상단 우측의 '저장시작'을 누르면 측정 데이터가 자동으로 저장됩니다.
  2. 측정이 끝나면 'REC' 버튼을 눌러 기록을 저장하고 측정을 종료합니다.
  3. 저장된 측정 데이터는 내 폰의 다운로드 폴더 > Signalwiz Mobile 폴더 안에 자동으로 저장됩니다.

저장된 데이터는 필요할 때 언제든지 확인할 수 있습니다.

또한 '임피던스' 탭에서 전극 접촉 상태를 확인하세요.
