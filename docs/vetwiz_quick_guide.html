VetWiz-ECG 퀵 가이드

VetWiz 기기 본체와 VetWiz 모바일 앱으로 동물의 심전도를 측정·저장하는 기본 순서를 한 페이지로 안내합니다. 자세한 내용은 상세 설명서와 어플 사용법을 참고하세요.

지원 OS: Android 10
시작 전 준비

① VetWiz 기기 본체를 USB Type-C로 충전합니다. (충전 중 LED 흰색, 완충 시 꺼짐)
② 구글 Play 스토어에서 ‘VetWiz’ 앱을 검색·설치하고 블루투스·알림 등 접근 권한을 허용합니다.
③ 스냅 전극 케이블(3개)과 3M 패치 전극을 준비합니다.

● 6단계로 끝내는 측정

  1. 전원 켜기 충전된 VetWiz 기기 본체의 전원을 켭니다.
  2. 전극 부착 SIG(좌측 뒷쪽 발바닥) · GND(좌측 앞쪽 발바닥) · REF(우측 앞쪽 발바닥) 위치에 전극을 발바닥과 밀착되도록 부착합니다.
  3. 앱 연결 VetWiz 모바일 앱을 실행하고 홈 화면에서 [기기 추가]를 누른 뒤 사용할 기기를 선택해 블루투스로 연결합니다. 연결에 성공하면 신호 파형이 나타납니다.
  4. 프로필 등록·선택 [프로필 등록 바로가기]에서 ‘QR코드’ 또는 ‘프로필 직접 등록하기’로 반려동물 프로필을 등록합니다.
  5. 측정(녹화) 모니터링 화면 우측 하단의 [녹화 시작]을 누르면 파형 녹화가 시작되고, [녹화 종료]를 누르면 측정 데이터가 저장됩니다.
  6. 결과 확인 [측정 데이터 리스트]에서 저장된 데이터를 선택해 상세 정보를 확인하고, [보고서 내보내기]로 PDF 보고서를 다운로드할 수 있습니다.
참고

측정 중에는 동물의 움직임을 최소화하고 전극이 떨어지지 않도록 고정하세요. 더 자세한 기능·문제 해결은 상세 설명서와 어플 사용법을 참고하시기 바랍니다.
