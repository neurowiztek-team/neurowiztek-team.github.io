VetWiz EEG 간편 안내

VetWiz EEG 기기 본체와 VetWiz EEG 앱으로 반려견 뇌파(EEG)를 측정·녹화하는 기본 순서를 한 페이지로 안내합니다. 사양·분석·리포트 등 자세한 내용은 상세 사용설명서를 참고하세요.

반려견 뇌파(EEG) 측정 8채널 · 250 SPS Android 10 이상

● 시작 전 준비

🧰

시작 전 준비

① VetWiz EEG 기기 본체를 USB Type-C로 충전합니다.
② 구글 Play 스토어에서 ‘VetWiz EEG’ 앱을 설치하고 블루투스·알림 권한을 허용합니다.
③ 골드컵 전극·전도성 페이스트·하이파픽스 테이프를 준비합니다.

📸

앱 화면 이미지 준비 중

아래 각 단계의 📸 표시 자리에 어떤 사진(기기·앱 화면)이 들어가야 하는지 표시해 두었습니다. 현재 VetWiz EEG 앱 화면 캡처를 추후 추가할 예정이며, 화면별 상세 지침은 /assets/images/vetwiz-eeg/manual/CAPTURES-NEEDED.md를 참고하세요. (SignalWiz 등 다른 앱 화면은 사용하지 않습니다.)

● 6단계로 끝내는 측정

  1. 전원 켜기 기기 본체의 전원 버튼을 한 번 터치하면 켜집니다. (끌 때는 약 3초간 길게 누릅니다.) 전극을 두피에 부착하고 케이블 연결 상태를 확인합니다.

    필요한 사진 — VetWiz EEG 기기 본체: 전면의 CH.1~CH.8·REF·GND 커넥터와 전원 버튼, 블루투스·충전 표시가 보이도록 촬영. (파일: 00_device_body.png)

  2. 앱 연결(블루투스) VetWiz EEG 앱을 실행하면 [블루투스] 화면이 열립니다. [기기 검색]으로 기기를 찾아 선택·연결하면 상단에 “연결됨”이 표시됩니다.

    필요한 사진 — 앱 [블루투스] 화면: 상단 “연결됨”, 등록된 기기 목록에서 기기가 연결된 상태. (파일: 01_bluetooth_connection.png)

  3. 채널 설정 [채널 설정] 탭에서 실제 전극을 부착한 채널(CH1~CH8)을 ‘사용’으로 선택하고 [설정 완료]를 누릅니다. 설정을 완료해야 임피던스·모니터를 사용할 수 있습니다.

    필요한 사진 — 앱 [채널 설정] 화면: CH1~CH8 사용/미사용 카드, ‘선택된 채널’ 목록, [설정 완료] 버튼. (파일: 02_channel_setup.png)

  4. 임피던스 확인 [임피던스] 탭에서 채널별 전극 접촉 상태를 확인합니다. 좋음일수록 양호하며, 매우 높음이면 전극을 다시 부착하고 전도성 페이스트를 보충합니다.

    필요한 사진 — 앱 [임피던스] 화면: CH1~CH8 각 채널의 임피던스 값(kΩ)과 좋음/높음/매우 높음 상태. (파일: 03_impedance.png)

  5. 실시간 모니터링 [모니터] 탭에서 선택한 채널의 EEG 파형을 실시간으로 확인합니다. 한 채널만 평평하거나 크게 튀면 전극·케이블 상태를 점검합니다.

    필요한 사진 — 앱 [모니터] 화면: 채널별 파형과 µV Y축, Time Window 5초, 상단 [녹화 시작] 버튼. (파일: 04_eeg_monitor.png)

  6. 녹화(저장) [모니터] 화면에서 녹화 시간을 정하고 우측 상단의 [녹화 시작]을 누른 뒤 피검체 정보(이름/ID·견종)를 입력하고 [측정 시작]을 누릅니다. [녹화 중지]로 종료하며, 저장 파일(CSV)은 다운로드 > VetWiz EEG 폴더에서 확인합니다.

    필요한 사진(2장) — 앱 [피검체 정보 입력] · [녹화 중] 화면: ① 정보 입력 다이얼로그(동물 이름/ID·견종·메모, [측정 시작]/[이름 없이 시작]), ② ‘녹화 중’ 상태와 [녹화 중지]. (파일: 08_subject_input.png · 09_recording.png)

📎

참고

녹화 중에는 전원 버튼으로 화면을 끄지 마세요(데이터 수신이 멈춥니다). 저장한 데이터의 파워스펙트럼·EEG Scalp Map 분석과 리포트(PDF)는 상세 사용설명서 — 분석을 참고하세요.
