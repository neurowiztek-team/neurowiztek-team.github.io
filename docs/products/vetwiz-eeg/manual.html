VetWiz EEG 사용설명서

VetWiz EEG는 반려견의 뇌파(EEG)를 측정하는 기기 본체와, 신호를 실시간으로 확인·녹화하고 정량 뇌파(qEEG)로 분석하는 VetWiz EEG 앱으로 구성됩니다. 이 문서는 처음 사용자를 위해 연결부터 측정·분석·리포트까지 한 곳에 담고 있습니다.

반려견 뇌파(EEG) 측정 8채널 · 250 SPS Android 10 이상

1 제품 개요

VetWiz EEG 기기 본체는 두피 전극으로 측정한 EEG 신호를 디지털로 변환하여 블루투스로 전송하고, VetWiz EEG 앱은 이 신호를 받아 전극 접촉(임피던스) 점검 · 실시간 파형 모니터링 · 녹화(CSV 저장) · 정량 분석(파워스펙트럼·EEG Scalp Map)을 수행합니다. 즉 기기는 측정·전송을, 앱은 표시·점검·저장·분석을 담당합니다.

기기와 앱의 역할

🧠

VetWiz EEG 기기 본체

8채널 EEG를 측정하여 블루투스로 앱에 실시간 전송하는 측정 기기입니다.

📱

VetWiz EEG 앱

연결·임피던스·모니터·녹화·분석·리포트를 수행하는 Android 앱입니다.

문서 적용 범위

  • VetWiz EEG 기기 본체와 VetWiz EEG 앱을 사용하는 운용자·연구자
  • 반려견의 EEG를 측정·기록하고, 파워스펙트럼·두피 분포·참고 지표를 확인하려는 사용자
⚠️

의료·진단 목적이 아닙니다

VetWiz EEG는 의료기기가 아니며 진단·치료 목적이 아닙니다. 측정·분석 결과는 qEEG 신호 특성을 정량화한 웰니스·교육·연구 참고 자료이며, 질환의 진단·판정·선별이 아닙니다. 최종 해석·진단은 담당 수의사가 임상검사·혈액검사·영상·설문과 종합하여 수행합니다.

2 안전을 위한 주의사항

⛔

경고 — 안전·기기 손상 관련

기기를 임의로 분해·개조하지 마세요. 물·습기·과도한 충격을 피하고, 부풀거나 발열·이상이 있는 배터리는 사용을 중단하세요. 충전 중에는 측정을 권장하지 않습니다.

⚠️

주의 — 측정 품질·데이터 관련

측정 중 반려견의 급격한 움직임은 동잡음(움직임 노이즈)을 유발합니다. 케이블이 당겨지거나 흔들리지 않도록 정리·고정하고, 강한 전기적 노이즈(전원 장치·모터 등) 근처를 피하세요. 녹화 중에는 전원 버튼으로 화면을 끄지 마세요(데이터 수신이 멈춥니다).

  • 측정 전 기기 배터리 잔량과 스마트폰 저장 공간을 확인합니다.
  • 전극과 소모품(전도성 페이스트·테이프)은 청결하게 관리하고, 손상된 케이블은 사용하지 않습니다.
  • 반려견이 불안해하면 무리하게 진행하지 말고 안정된 상태에서 측정합니다.
  • 측정 부위 피부에 상처·발진이 있으면 전극을 부착하지 않습니다.
  • 사용 후에는 전극에 남은 페이스트를 닦아내고 건조하여 보관합니다.

4 기기 각부 명칭

VetWiz EEG 기기 본체 전면에는 전극 커넥터와 전원 버튼, 상태 표시가 배치되어 있습니다.

📸

기기 본체 사진 준비 중

이 자리에는 VetWiz EEG 기기 본체를 직접 촬영한 사진이 들어갑니다. 전면의 CH.1~CH.8·REF·GND 전극 커넥터, 전원 버튼, 블루투스·충전 상태 표시가 보이도록 촬영하세요. SignalWiz 등 다른 제품 문서의 사진을 재사용하지 않습니다. (촬영 지침: /assets/images/vetwiz-eeg/manual/CAPTURES-NEEDED.md)

기기 각부 명칭
CH.1 ~ CH.8 8개 측정 채널의 전극 커넥터
REF(기준 전극) 기준(Reference) 전극 커넥터 — 이 기기는 기준(REF)이 후두부(Occiput)로 고정입니다.
GND(접지 전극) 접지(Ground) 전극 커넥터 — 이 기기는 접지(GND)가 비근부(Nasion)로 고정입니다.
전원 버튼(⏻) 한 번 터치하면 켜지고, 약 3초간 길게 누르면 꺼집니다.
상태 표시 본체 하단의 블루투스(✳)·충전(⚡) 표시 아이콘
🧭

기준·접지 전극은 고정입니다

기준(REF)=후두부(Occiput), 접지(GND)=비근부(Nasion)로 고정되어 있어 사용자가 변경할 수 없습니다. CH.1~CH.8의 전극 위치(10-20 구성)는 앱의 [채널 설정]에서 확인·조정할 수 있습니다.

6 사용 전 준비

  1. 기기 충전측정 전에 USB Type-C 케이블로 기기 본체를 충분히 충전합니다. 완충 시 약 13시간 연속 사용할 수 있으며, 장시간 측정 전에는 배터리 잔량을 미리 확인합니다.
  2. 앱 설치구글 Play 스토어에서 ‘VetWiz EEG’ 앱을 설치합니다. (Android 10 이상 기기 권장)
  3. 권한 허용앱을 처음 실행하면 블루투스 검색·연결 권한 요청이 표시됩니다. 모두 허용합니다. Android 11 이하에서는 기기 검색을 위해 위치 권한도 필요합니다.
  4. 알림 권한 허용녹화는 백그라운드 서비스로 동작하므로, 측정 중 상태 알림 표시를 위해 알림 권한을 허용합니다.
  5. 전극·소모품 준비골드컵 전극을 본체 단자에 연결하고 전도성 페이스트와 하이파픽스 의료용 테이프를 준비합니다.
  6. 측정 환경 준비전기적 노이즈가 적은 조용한 환경에서 측정하고, 스마트폰과 기기를 가까이 둡니다. 반려견이 안정될 수 있는 자세를 준비합니다.
  7. 피검체 정보 준비측정할 반려견의 이름/ID와 견종을 준비합니다. 녹화 시작 시 입력하며, 견종은 같은 개체의 종단 추세를 묶는 기준으로 사용됩니다.
💾

저장 권한 안내

녹화 데이터는 안드로이드 공용 다운로드 폴더의 VetWiz EEG 폴더에 저장되며, 별도의 파일 저장 권한을 요구하지 않습니다(Android 10 이상 범위 저장소 방식).

7 앱 시작 화면

앱을 실행하면 잠깐 시작 화면(“기억이 생명이다”)이 표시된 뒤, 가장 먼저 [블루투스] 화면이 열립니다. 처음 사용 시에는 화면을 짚어 주는 사용 안내(튜토리얼)가 함께 표시됩니다.

8 기기 연결(블루투스)

앱을 실행하면 [블루투스] 화면이 표시됩니다. 아래 순서대로 VetWiz EEG 기기 본체를 연결합니다.

📸

앱 화면 새로 캡처 필요 — [블루투스]

이 자리에는 현재 VetWiz EEG 앱의 블루투스 화면이 들어갑니다: 상단 “연결됨”, 등록된 기기 목록에서 기기가 연결된 상태. SignalWiz 등 다른 앱 화면을 사용하지 않습니다. (촬영 지침: /assets/images/vetwiz-eeg/manual/CAPTURES-NEEDED.md)

  1. 기기 전원 켜기본체의 전원 버튼을 한 번 터치하면 전원이 켜집니다. (끌 때는 전원 버튼을 약 3초간 길게 누릅니다.)
  2. 앱 실행 및 권한 허용앱을 실행하면 [블루투스] 화면이 표시됩니다. 권한 요청이 나타나면 모두 허용합니다.
  3. 기기 검색[기기 검색]을 눌러 주변 기기를 검색합니다. 이미 등록된 기기는 등록된 기기 목록에, 새로 검색된 기기는 검색된 기기 목록에 표시됩니다.
  4. 기기 선택목록에서 사용할 VetWiz EEG 기기 본체를 선택합니다. 처음 사용하는 기기는 [등록]으로 등록한 뒤 연결합니다.
  5. 연결 완료 확인연결되면 화면 상단 상태가 “연결됨”으로 표시되고 기기 이름이 함께 나타납니다. (연결 진행 중에는 “연결 중”이 표시됩니다.)
  6. 다음 단계로 이동연결이 완료되면 하단 탭에서 [채널 설정]으로 이동하여 측정 채널을 지정합니다.
🔌

연결이 되지 않거나 끊길 때

기기가 목록에 없으면 본체 전원과 권한 허용 여부를 먼저 확인하세요. 연결에 실패하면 “기기에 연결하지 못했습니다. 기기 전원과 거리를 확인한 뒤 잠시 후 다시 시도해 주세요.” 안내가 표시됩니다. 측정 중 연결이 끊기면 “블루투스 연결 끊김” 안내가 표시되며 [블루투스] 화면으로 이동합니다. 전원·거리를 확인한 뒤 다시 연결하세요.

9 측정 채널 설정

실제 전극을 부착(사용)한 채널만 선택하여 측정·저장 대상으로 지정하는 단계입니다. 채널 설정을 완료해야 임피던스 측정과 데이터 녹화를 사용할 수 있습니다.

📸

앱 화면 새로 캡처 필요 — [측정 채널 설정]

이 자리에는 현재 VetWiz EEG 앱의 채널 설정 화면이 들어갑니다: CH1~CH8 사용/미사용 카드, 선택된 채널 목록, [설정 완료] 버튼. SignalWiz 등 다른 앱 화면을 사용하지 않습니다. (촬영 지침: /assets/images/vetwiz-eeg/manual/CAPTURES-NEEDED.md)

  1. [채널 설정] 탭 이동하단 탭에서 [채널 설정]을 선택합니다.
  2. 사용 채널 선택CH1~CH8 카드 중 실제 전극을 부착한 채널을 눌러 ‘사용’ 상태로 설정합니다. 사용하지 않는 채널은 ‘미사용’으로 둡니다.
  3. 선택 내용 확인화면 하단에서 선택된 채널 목록을 확인합니다.
  4. 설정 완료[설정 완료] 버튼을 누르면 선택한 채널 정보가 기기에 전달되고, 임피던스 측정과 녹화 기능이 활성화됩니다. 채널을 바꾼 뒤에는 [설정 완료]를 다시 눌러야 변경 내용이 적용됩니다.
🧩

전극 위치(10-20 구성)

채널 설정 화면에서는 각 채널의 전극 위치(EEG 10-20 구성)를 확인·조정할 수 있습니다. 기본 8채널 구성은 Fp1, Fp2, F3, F4, C3, C4, O1, O2이며, 프리셋(표준·전두 강조·측두 강조·후두 강조)으로 빠르게 바꿀 수 있습니다. 기준(REF)·접지(GND)는 후두부·비근부로 고정입니다.

10 임피던스 확인

임피던스 측정은 전극이 두피에 얼마나 잘 접촉되어 있는지를 확인하는 과정입니다. 값이 낮고 안정적일수록 전극 접촉 상태가 양호하며 신호 품질도 좋아집니다.

📸

앱 화면 새로 캡처 필요 — [임피던스]

이 자리에는 현재 VetWiz EEG 앱의 임피던스 화면이 들어갑니다: CH1~CH8 각각의 임피던스 값(kΩ)과 좋음/높음/매우 높음 상태. SignalWiz 등 다른 앱 화면을 사용하지 않습니다. (촬영 지침: /assets/images/vetwiz-eeg/manual/CAPTURES-NEEDED.md)

  1. 전극 부착 확인측정 부위에 전극을 부착하고 케이블 연결 상태를 확인합니다. 필요 시 전도성 페이스트를 보충합니다.
  2. [임피던스] 탭 이동하단 탭에서 [임피던스]를 선택하면 측정이 자동으로 시작됩니다.
  3. 순차 측정 확인임피던스는 채널 단위로 순차 측정됩니다. 화면에는 “현재 측정 중: CHn”과 진행 상황(측정 진행: n / 8)이 표시됩니다.
  4. 채널별 값 확인각 채널 카드에 임피던스 값(kΩ)과 상태(좋음/높음/매우 높음)가 표시됩니다.
  5. 접촉 불량 조치값이 높거나 ‘매우 높음’으로 표시되는 채널은 전극을 다시 부착하고 전도성 페이스트를 보충한 뒤 접촉 상태를 안정화합니다.

참고

위 구간 값은 앱의 상태 표시 기준이며, 실제 측정 환경·전극 종류에 따라 양호 기준은 달라질 수 있습니다. 선택한 채널만 순차적으로 측정되며, 녹화 시작 시점의 임피던스 값은 저장 파일에도 함께 기록됩니다.

11 실시간 EEG 모니터링

[모니터](신호 모니터) 화면에서는 현재 수신 중인 EEG 신호를 채널별 파형으로 실시간 확인할 수 있습니다.

📸

앱 화면 새로 캡처 필요 — [신호 모니터]

이 자리에는 현재 VetWiz EEG 앱의 신호 모니터 화면이 들어갑니다: 상단 시간 설정과 [녹화 시작], Time Window 5초, 채널별 파형과 µV Y축. SignalWiz 등 다른 앱 화면을 사용하지 않습니다. (촬영 지침: /assets/images/vetwiz-eeg/manual/CAPTURES-NEEDED.md)

화면 구성

  • 채널별 파형 카드 — 채널 설정에서 선택한 채널만 CH n 라벨과 색상으로 표시됩니다.
  • 진폭(Y) 축 — 세로축은 신호 진폭(µV)이며, 기본 표시 범위는 −100µV ~ +100µV입니다. 카드 왼쪽의 축(µV) 영역을 누르면 모든 채널의 표시 범위를 함께 바꿀 수 있습니다.
  • 시간(X) 축 — 가로축은 시간으로, 최근 약 5초(Time Window: 5 s) 구간의 파형이 흐르듯 표시됩니다.
  • 연결 상태 — 데이터 수신 시 “연결됨”, 신호가 없으면 “신호 없음”이 표시됩니다.
  • 녹화 시작 버튼 — 화면 우측 상단의 [녹화 시작]으로 녹화를 시작/중지하고, 그 옆에서 녹화 시간을 설정합니다.
〰️

표시되는 신호 안내

모니터 파형은 기저선 흔들림과 잡음을 줄이기 위해 0.5~50 Hz 밴드패스와 60 Hz·31.2 Hz 노치 필터가 적용된 신호입니다. 이 필터는 항상 적용되며 사용자가 끄거나 값을 바꿀 수 없습니다.

트리거 표시(선택)

녹화 중 [시작 트리거]·[끝 트리거]를 누르면 그 시점이 저장되는 CSV 데이터에 함께 기록됩니다. 앱은 트리거 구간을 자동 분석하지 않으며, 트리거를 활용한 추가 분석은 저장된 데이터로 직접 진행합니다.

🔆

주의

녹화 중에는 전원 버튼으로 화면을 끄지 마세요. 화면이 꺼지면 데이터 수신이 멈춥니다. 한 채널만 평평하거나 크게 튀면 해당 채널의 전극 접촉·케이블 연결을 확인하고, 필요하면 [임피던스] 화면에서 다시 점검합니다.

12 측정 및 녹화

측정 신호는 [모니터] 화면에서 녹화하여 CSV 파일로 저장합니다. 녹화는 백그라운드 서비스로 동작하므로 안정적으로 기록됩니다.

  1. 녹화 시간 설정[모니터] 화면에서 녹화 시간을 선택합니다. 10초·30초·1분·3분·5분·10분·30분·1시간 중에서 고르거나 무제한(직접 종료까지 녹화)으로 둘 수 있습니다. 시간을 지정하면 시간이 끝날 때 자동으로 저장됩니다.
  2. 녹화 시작우측 상단의 [녹화 시작]을 누르면 피검체 정보 입력 창이 나타납니다. 측정할 동물의 이름/ID와 견종·메모를 입력하고 [측정 시작]을 누릅니다. 이름 없이 진행하려면 [이름 없이 시작]을 선택합니다.
  3. 녹화 중 상태녹화가 시작되면 버튼이 [녹화 중지]로 바뀌고 상태가 ‘녹화 중’으로 표시되며, 상태 표시줄에 “데이터 기록 중” 알림이 나타납니다. 녹화 중에는 다른 탭으로 이동할 수 없습니다.
  4. 녹화 종료설정한 시간이 지나면 자동으로 저장되고, [녹화 중지]를 누르면 직접 종료할 수 있습니다.
  5. 저장 확인저장이 끝나면 “녹화 저장됨 · … · 분석 탭에서 확인·수정할 수 있습니다” 안내가 표시됩니다. 파일은 스마트폰의 다운로드 > VetWiz EEG 폴더에 저장됩니다.

저장 파일 안내

저장 파일은 CSV 형식이며 파일명은 Raw로 시작합니다. 파일 첫 부분에는 측정 시작·종료 시각, 샘플링 주파수, 채널 수, 선택 채널, 녹화 전 임피던스 등 측정 정보가 함께 기록됩니다. 컬럼은 Timestamp, SampleIndex, Ch1~Ch8, aux1~aux3, Trigger 정보로 구성되며, 선택하지 않은 채널은 0으로 기록됩니다. 피검체 정보 등 세션 정보는 .meta.json 파일로 함께 저장됩니다.

13 녹화 기록 관리

저장된 녹화 파일은 [분석] 탭의 [녹화 CSV 불러오기]를 통해 열리는 녹화 라이브러리에서 관리합니다.

  • 목록·검색 — 저장된 녹화가 목록으로 표시되며, 동물 이름·견종·메모가 함께 보입니다. 상단의 이름으로 검색으로 찾을 수 있습니다.
  • 불러오기 — 파일을 선택하면 분석 화면으로 불러옵니다.
  • 피검체 정보 수정 — 저장된 파일의 피검체 정보(이름·견종·메모)를 나중에 수정할 수 있습니다.
  • 공유 — [공유]로 CSV 파일을 다른 앱으로 전송합니다.
  • 삭제 — [삭제]로 파일을 삭제합니다. 삭제는 되돌릴 수 없습니다.

데이터 폴더

기본 데이터 폴더는 다운로드 > VetWiz EEG입니다. 앱을 재설치했거나 다른 기기에서 파일을 옮긴 경우, [설정] → 데이터 불러오기 경로에서 폴더를 다시 선택해 접근을 허용하면 기존 파일을 계속 사용할 수 있습니다. 피검체 정보는 기기 내부에만 저장되므로 재설치 전 [설정]에서 백업(내보내기)을 권장합니다.

14 분석(스펙트럼·Scalp Map)

[분석] 탭에서 녹화 CSV를 불러오면 채널별 파워스펙트럼과 EEG Scalp Map, 참고 지표를 확인할 수 있습니다.

📸

앱 화면 새로 캡처 필요 — [분석] (2장)

이 자리에는 현재 VetWiz EEG 앱의 분석 화면 2장이 들어갑니다. ① 파워스펙트럼 — [녹화 CSV 불러오기], 파일 정보 카드(녹화 시간·250 SPS·0–50 Hz·채널 수), 채널 선택, 파워스펙트럼 그래프. ② EEG Scalp Map — Delta/Theta/Alpha/Beta/Gamma 대역 선택, 강아지 두피 분포 색상 지도와 0~100% 색상 막대. (파워스펙트럼·Scalp Map은 SignalWiz 앱에 없는 VetWiz EEG 전용 기능입니다. 촬영 지침: /assets/images/vetwiz-eeg/manual/CAPTURES-NEEDED.md)

  1. 녹화 CSV 불러오기[녹화 CSV 불러오기]를 눌러 녹화 라이브러리에서 분석할 파일을 선택합니다.
  2. 파일 정보 확인불러온 파일의 녹화 시간, 샘플링 주파수(250 SPS), 분석 대역(0–50 Hz), 채널 수, 유효(Clean) 구간 비율이 요약으로 표시됩니다.
  3. 채널 선택[전체 선택]·[전체 해제] 또는 CH1~CH8을 눌러 분석할 채널을 지정합니다. 이 녹화에서 측정되지 않은 채널은 선택할 수 없습니다.
  4. 파워스펙트럼 확인파워스펙트럼 그래프에서 주파수(Hz)에 따른 파워(dB)를 채널별로 확인합니다.
  5. 두피 분포 확인EEG Scalp Map에서 대역(Delta·Theta·Alpha·Beta·Gamma)을 선택해 각 채널의 상대 대역 파워 분포를 두피 지도로 확인합니다.

지표 해석 주의

이 지표들은 qEEG 신호 특성을 정량화한 참고 자료이며 질환의 진단·판정·선별이 아닙니다. 알파 기반 지표(이완/균형/전두엽)는 눈을 감은 안정 상태에서 1~3분 측정 시 가장 신뢰할 수 있습니다. 움직임 노이즈가 많으면 “분석 신뢰도가 낮을 수 있습니다” 안내가 표시되므로, 안정된 상태에서 다시 측정하는 것을 권장합니다.

15 분석 보고서(PDF)

분석 결과는 여러 페이지로 구성된 정량 뇌파(qEEG) 리포트(PDF)로 내보낼 수 있습니다.

  1. 분석 파일 준비[분석] 탭에서 리포트로 만들 녹화 파일을 불러오고 채널·대역을 확인합니다.
  2. 보고서 출력[분석 보고서 출력]을 누릅니다. 보고서 PDF는 측정 파일당 1회 결제 상품이며, 한 번 결제한 파일은 재결제 없이 다시 생성할 수 있습니다.
  3. PDF 생성·저장생성이 완료되면 PDF가 열리며, 파일은 다운로드 > VetWiz EEG 폴더에 저장됩니다.
  4. 공유생성한 리포트는 이메일 등으로 공유할 수 있습니다.

리포트 구성

  • 1페이지 종합 요약(기본 정보·상태 요약·3축 판독·소견·권고·안내)
  • 대표 파형, 두피맵(Scalp Map), 파워스펙트럼(PSD)
  • 결합도·복잡도, 지표 상세, 참고문헌
🩺

리포트 필수 안내

본 리포트는 의료기기가 아니며, qEEG 신호 특성의 측정·정량화일 뿐 질환 진단·판정·선별이 아닙니다. 최종 해석·진단은 담당 수의사가 임상검사·혈액검사·영상·설문과 종합하여 수행합니다.

💳

결제 관련 안내

결제가 승인 대기 중이면 “결제가 승인 대기 중입니다. 승인이 완료되면 다시 시도해 주세요.” 안내가 표시됩니다. PDF를 열 수 있는 앱이 없으면 PDF 뷰어를 설치한 뒤 다시 확인하세요.
