SignalWiz 간편 안내

SignalWiz 기기 본체와 SignalWiz 모바일 앱으로 EEG를 측정·저장하는 기본 순서를 한 페이지로 안내합니다. 자세한 내용은 상세 제품 사용설명서를 참고하세요.

지원 OS: Android 10 이상
시작 전 준비

① SignalWiz 기기 본체를 USB Type-C로 충전합니다.
② 구글 Play 스토어에서 ‘Signalwiz mobile’ 앱을 검색·설치하고 블루투스·알림 권한을 허용합니다.
③ 골드컵 전극·전도성 페이스트·하이파픽스 테이프를 준비합니다.

● 6단계로 끝내는 측정

  1. 전원 켜기 SignalWiz 기기 본체의 전원 버튼을 한 번 터치하면 켜집니다. (끌 때는 약 3초간 길게 누릅니다.)
  2. 앱 연결 SignalWiz 모바일 앱을 실행하고 [블루투스] 화면에서 기기를 선택해 연결합니다. 상단에 “연결됨”이 표시되면 완료입니다.
  3. 채널 설정 [채널 설정] 탭에서 전극을 부착한 채널을 선택하고 [설정 완료]를 누릅니다. 설정을 완료해야 임피던스·모니터를 사용할 수 있습니다.
  4. 임피던스 확인 [임피던스] 탭에서 채널별 전극 접촉 상태를 확인합니다. GOOD(초록)일수록 양호하며, OPEN(빨강)이면 전극을 다시 부착하고 전도성 페이스트를 보충합니다.
  5. 실시간 모니터링 [모니터] 탭에서 선택한 채널의 EEG 파형을 실시간으로 확인합니다. 특정 채널만 평평하거나 크게 튀면 전극·케이블 상태를 점검합니다.
  6. 저장(녹화) [모니터] 화면 우측 상단의 [저장 시작]을 누르면 녹화가 시작되고, [저장 중지]로 종료합니다. 저장된 CSV는 Download/Signalwiz mobile/ 폴더에서 확인합니다.
