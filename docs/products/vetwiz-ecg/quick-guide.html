VetWiz-ECG 간편 안내

VetWiz-ECG 기기와 VetWiz 앱으로 반려동물의 생체 신호를 측정·저장하는 기본 순서를 한 페이지로 안내합니다. 화면별 상세 설명과 기기·전극·충전은 상세 사용설명서를 참고하세요.

시작 전 준비

① VetWiz-ECG 기기 본체와 스냅 전극 케이블·패치 전극을 준비합니다.
② 구글 Play 스토어에서 ‘VetWiz’ 앱을 검색·설치하고 블루투스·알림 권한을 허용합니다.
③ 블루투스를 켠 스마트폰을 기기 가까이에 둡니다.

● 6단계로 끝내는 측정

  1. 앱 설치 및 권한 허용 구글 Play 스토어에서 VetWiz 앱을 설치하고, 실행 후 블루투스·알림 등 접근 권한을 허용합니다.
  2. 기기 연결 홈 화면에서 [기기 추가]를 누르고 사용할 기기를 선택해 블루투스로 연결합니다. 연결에 성공하면 실시간 생체 신호(파형)가 나타납니다.
  3. 프로필 등록 [프로필 등록 바로가기]에서 ‘QR코드’ 또는 ‘프로필 직접 등록하기’로 반려동물 프로필을 등록합니다.
  4. 프로필 선택 모니터링 화면으로 이동한 뒤 좌측 상단의 [프로필을 선택하여 주세요]를 눌러 측정할 반려동물을 선택합니다.
  5. 측정(녹화) 우측 하단의 [녹화 시작]을 누르면 생체 신호가 녹화됩니다. [녹화 종료]를 누르면 측정 데이터가 저장되고 레포트 생성이 진행됩니다.
  6. 결과 확인 · 보고서(PDF) 측정 데이터 리스트에서 저장된 데이터를 선택해 상세 정보를 확인하고, 우측 하단의 [보고서 내보내기]로 PDF 파일을 생성·다운로드할 수 있습니다.
